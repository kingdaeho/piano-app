## 1. 정보 구조 (Information Architecture)

### 1.1 전체 네비게이션 구조

```
Piano Practice Companion
│
├── [Tab 1] 홈 (Home)
│   ├── 오늘의 대시보드
│   │   ├── 목표 달성률 (프로그레스 링)
│   │   ├── 연습 스트릭
│   │   ├── 빠른 연습 시작 버튼 (FAB)
│   │   └── 오늘 연습 세션 요약
│   ├── 최근 레슨 노트 (카드)
│   ├── 현재 연습 중인 곡 (카드 리스트)
│   └── 주간 연습 미니 차트
│
├── [Tab 2] 연습 (Practice)
│   ├── 연습 타이머 화면
│   │   ├── 타이머 (시작/일시정지/종료)
│   │   ├── 현재 연습 곡 선택
│   │   ├── 곡 전환 (다른 곡으로 변경)
│   │   └── 연습 메모 입력
│   ├── 연습 히스토리
│   │   ├── 일별 목록
│   │   ├── 주별 요약
│   │   └── 세션 상세 보기
│   └── 연습 완료 요약
│       ├── 총 연습 시간
│       ├── 곡별 시간 분배
│       └── 간단 자기평가 (이모지)
│
├── [Tab 3] 레슨 (Lessons)
│   ├── 레슨 노트 목록
│   │   ├── 회차순 정렬
│   │   ├── 검색 (곡명, 키워드)
│   │   └── 필터 (기간, 곡)
│   ├── 레슨 노트 작성
│   │   ├── 회차/날짜 (자동)
│   │   ├── 레슨 내용
│   │   ├── 선생님 피드백
│   │   ├── 과제 체크리스트
│   │   ├── 연관 곡 태그
│   │   └── 이미지 첨부
│   └── 레슨 노트 상세
│       ├── 내용 보기
│       ├── 과제 체크 상태
│       └── 연관 연습 기록
│
├── [Tab 4] 곡 관리 (Repertoire)
│   ├── 곡 목록
│   │   ├── 상태별 필터 (전체/연습 중/완성/보류)
│   │   ├── 난이도별 필터
│   │   └── 정렬 (최근 연습순/등록순/난이도순)
│   ├── 곡 등록
│   │   ├── 곡명, 작곡가
│   │   ├── 난이도 (1~5)
│   │   ├── 장르/교재
│   │   └── 진행 상태
│   └── 곡 상세
│       ├── 기본 정보
│       ├── 진행도 (%)
│       ├── 누적 연습 시간
│       ├── 구간별 메모
│       └── 관련 레슨 노트
│
├── [Tab 5] 통계 (Stats)
│   ├── 주간/월간 연습 통계
│   │   ├── 총 연습 시간
│   │   ├── 일별 막대 차트
│   │   └── 곡별 연습 비율
│   ├── 캘린더 히트맵
│   │   └── 월별 연습량 시각화

│   └── 타임라인
│       ├── 마일스톤 이벤트
│       └── 학습 여정 뷰
│
└── [더보기/설정]
    ├── 목표 설정
    │   ├── 일일 목표 시간
    │   ├── 주간 목표 일수/시간

    ├── 프로필
    ├── 알림 설정
    ├── 데이터 백업/내보내기
    └── 앱 설정 (다크모드, 언어)
```

### 1.2 탭 구성 요약

| 탭 | 아이콘 | 역할 | 핵심 액션 |
|----|--------|------|-----------|
| 홈 | `house` | 오늘의 현황 한눈에 | 빠른 연습 시작 |
| 연습 | `timer` | 연습 기록/타이머 | 타이머 시작/종료 |
| 레슨 | `book` | 레슨 노트 관리 | 노트 작성/조회 |
| 곡관리 | `music.note.list` | 곡 목록/진행도 | 곡 등록/상태 변경 |
| 통계 | `chart.bar` | 통계/타임라인 | 데이터 조회 |

### 1.3 화면 계층 관계

```
Level 0 (탭)       Level 1 (메인)         Level 2 (상세/작성)      Level 3 (서브)
─────────────      ───────────────        ──────────────────      ──────────────
홈 ──────────────→ 대시보드
                   ├──────────────────→ 연습 시작 (타이머)
                   ├──────────────────→ 레슨 노트 상세
                   └──────────────────→ 곡 상세

연습 ────────────→ 연습 히스토리
                   ├──────────────────→ 연습 타이머 ──────────→ 곡 선택 모달
                   │                                          → 연습 완료 요약
                   └──────────────────→ 세션 상세

레슨 ────────────→ 레슨 노트 목록
                   ├──────────────────→ 노트 작성 ────────────→ 곡 태그 선택
                   │                                          → 이미지 첨부
                   └──────────────────→ 노트 상세 ────────────→ 과제 체크

곡관리 ──────────→ 곡 목록
                   ├──────────────────→ 곡 등록/편집
                   └──────────────────→ 곡 상세 ──────────────→ 구간별 메모
                                                              → 연습 기록

통계 ────────────→ 통계 대시보드
                   ├──────────────────→ 캘린더 히트맵
                   ├──────────────────→ 목표 설정/관리
                   └──────────────────→ 타임라인
```

### 2.5 레슨 노트 작성 화면

│ ◀ 상태바                          9:41 │
├─────────────────────────────────────────┤
│  ✕ 닫기       레슨 노트 작성      저장  │
│                                         │
│ ┌─────────────────────────────────────┐ │
│ │  회차     │ 13회차 (자동)           │ │
│ │  날짜     │ 2026.03.19 (수) ▼      │ │
│ │  시간     │ 14:00 ~ 14:50          │ │

│                                         │

│ ┌─────────────────────────────────────┐ │

## 5. 반응형 고려사항

### 5.1 모바일 (Phone) - 기본

모든 화면이 모바일 우선으로 설계됩니다.

| 항목 | 사양 |
|------|------|
| **화면 너비** | 375~430pt (iPhone 기준) |
| **레이아웃** | 단일 컬럼 |
| **네비게이션** | 하단 탭 바 (5탭) |
| **카드** | 전체 너비 (좌우 패딩 20px) |
| **리스트** | 수직 스크롤 |
| **차트** | 전체 너비, 터치 인터랙션 |
| **타이머** | 중앙 배치, 큰 폰트 (Display) |
| **모달** | 바텀 시트 (하단에서 올라옴) |
| **터치 타겟** | 최소 44x44pt |

### 5.2 태블릿 (iPad/Android Tablet) - 적응형

태블릿에서는 넓은 화면을 활용하여 정보 밀도를 높입니다.

| 항목 | 모바일 | 태블릿 차이점 |
|------|--------|--------------|
| **화면 너비** | 375~430pt | 768~1024pt |
| **레이아웃** | 단일 컬럼 | 2컬럼 (마스터-디테일) 또는 그리드 |
| **네비게이션** | 하단 탭 바 | 좌측 사이드바 (Rail 또는 Drawer) |
| **홈 대시보드** | 수직 스크롤 카드 | 2컬럼 그리드, 위젯 스타일 배치 |
| **연습 타이머** | 전체 화면 | 좌: 타이머/곡 정보, 우: 메모/곡 목록 |
| **레슨 노트 목록** | 전체 화면 리스트 | 좌: 목록, 우: 상세 (스플릿 뷰) |
| **곡 관리** | 수직 리스트 | 2~3컬럼 그리드 카드 |
| **통계** | 스크롤 차트 | 2컬럼 대시보드, 차트 병렬 배치 |
| **모달** | 바텀 시트 | 중앙 다이얼로그 또는 팝오버 |
| **카드 너비** | 전체 너비 | 최대 480px, 여백 자동 조정 |

### 5.3 Safe Area 및 시스템 UI 대응

| 항목 | iOS | Android |
|------|-----|---------|
| 상단 안전 영역 | Dynamic Island / Notch | Status Bar |
| 하단 안전 영역 | Home Indicator (34pt) | Navigation Bar (Gesture/3-Button) |
| 키보드 | 자동 스크롤, 입력 필드 위로 이동 | 동일 |
| 가로 모드 | 지원 (레이아웃 적응) | 지원 |

### 5.4 접근성 고려

| 항목 | 가이드라인 |
|------|-----------|
| 최소 터치 타겟 | 44x44pt (iOS) / 48x48dp (Android) |
| 색상 대비 | WCAG AA: 일반 텍스트 4.5:1, 큰 텍스트 3:1 |
| 다이나믹 타입 | 시스템 폰트 크기 설정 반영 (최대 2배) |
| VoiceOver/TalkBack | 모든 인터랙티브 요소에 접근성 레이블 |
| 모션 감소 | `prefers-reduced-motion` 설정 시 애니메이션 최소화 |
