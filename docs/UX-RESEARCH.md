## 1. 사용자 페르소나

### 페르소나 1: 독학 입문자 - 김수진 (29세, 직장인)

| 항목 | 내용 |
|------|------|
| **이름** | 김수진 |
| **나이** | 29세 |
| **직업** | IT 회사 마케팅 담당 |
| **피아노 경험** | 전무 (완전 초보) |
| **사용 기기** | iPhone 15, iPad Air |

**배경**
- 어릴 때부터 피아노를 배우고 싶었지만 기회가 없었음
- 최근 디지털 피아노(Yamaha P-125)를 충동 구매
- 유튜브로 독학 시작했지만 체계적이지 않아 막막함
- 퇴근 후 30분~1시간 연습 시간 확보 가능

**목표**
- 좋아하는 팝송 3곡을 6개월 안에 연주하고 싶음
- 기초 코드와 음계를 탄탄히 익히고 싶음
- 꾸준히 연습하는 습관을 만들고 싶음

**좌절점 (Pain Points)**
- 유튜브 영상이 너무 많아 어디서부터 시작해야 할지 모름
- 연습을 하다가 며칠 건너뛰면 어디까지 했는지 잊어버림
- 혼자 배우다 보니 내가 제대로 하고 있는지 확신이 없음
- 연습 시간이 불규칙하고, 동기부여가 떨어지는 순간이 자주 옴

**앱 사용 시나리오**
1. 퇴근 후 앱을 열어 오늘의 연습 목표 확인
2. 어제 연습한 곡의 특정 구간을 이어서 연습
3. 연습 시간과 곡별 진도를 기록
4. 주간 연습 통계를 보며 성취감을 느낌
5. 다음에 연습할 곡과 구간을 메모로 남김

### 페르소나 2: 레슨 수강생 - 박민호 (35세, 프리랜서)

| 항목 | 내용 |
|------|------|
| **이름** | 박민호 |
| **나이** | 35세 |
| **직업** | 프리랜서 디자이너 |
| **피아노 경험** | 6개월차 (주 1회 레슨) |
| **사용 기기** | Galaxy S24, 갤럭시 탭 |

**배경**
- 동네 피아노 학원에서 주 1회 레슨 수강 중
- 선생님이 매주 과제를 내주지만, 종이 노트에 적으면 잃어버리기 일쑤
- 레슨 때 선생님이 말해준 포인트를 기억하기 어려움
- 레슨비를 내는 만큼 효율적으로 배우고 싶음

**목표**
- 선생님이 내준 과제를 빠짐없이 연습하고 싶음
- 레슨에서 배운 핵심 내용을 체계적으로 정리하고 싶음
- 연습량을 늘려서 레슨 진도를 더 빨리 나가고 싶음
- 1년 안에 쇼팽 왈츠 한 곡 완주가 목표

**좌절점 (Pain Points)**
- 선생님이 레슨 중 알려준 세부 피드백을 기억 못함 (템포, 손가락 번호, 강약 등)
- 종이 노트에 적어도 정리가 안 되고, 회차별 추적이 안 됨
- 어떤 곡의 어떤 마디를 집중 연습해야 하는지 한눈에 보이지 않음
- 레슨 전 복습할 때 지난 레슨 내용을 찾기 번거로움

**앱 사용 시나리오**
1. 레슨 직후: 선생님 피드백을 회차별로 앱에 기록 (곡명, 마디, 주의점)
2. 매일 연습: 오늘 연습할 과제 목록 확인 후 연습 시작
3. 연습 기록: 곡별 연습 시간과 구간별 완성도 체크
4. 레슨 전: 이번 주 연습 기록과 지난 레슨 노트를 복습
5. 월별 리뷰: 레슨 히스토리와 연습 통계로 성장 확인

### 페르소나 3: 피아노 재개자 - 이은주 (42세, 주부)

| 항목 | 내용 |
|------|------|
| **이름** | 이은주 |
| **나이** | 42세 |
| **직업** | 전업주부 (두 자녀의 엄마) |
| **피아노 경험** | 초등~중학교 5년간 배움, 20년 이상 공백 |
| **사용 기기** | iPhone 14, iPad |

**배경**
- 어릴 때 체르니 40번까지 진도를 나간 경험
- 아이들 피아노 학원 보내면서 다시 피아노에 관심이 생김
- 집에 아이들 디지털 피아노가 있어 접근이 용이
- 아이들이 학교 간 오전 시간에 틈틈이 연습 가능

**목표**
- 예전에 쳤던 곡들을 다시 연주하고 싶음
- 아이들 학원 발표회에서 엄마가 반주해주는 것이 꿈
- 매일 꾸준히 연습하되, 자기만의 속도로 부담 없이
- 새로운 장르(재즈, 뉴에이지)에도 도전해보고 싶음

**좌절점 (Pain Points)**
- 예전에 쉽게 했던 것들이 손가락이 굳어서 안 됨 (좌절감)
- 어디서부터 다시 시작해야 할지 레벨 판단이 어려움
- 독학과 레슨 사이에서 고민 (레슨 받기엔 시간이 불규칙)
- 연습 기록을 남기고 싶지만 복잡한 앱은 거부감이 있음

**앱 사용 시나리오**
1. 아침: 앱에서 연습 리마인더 알림 확인
2. 연습 시작: 현재 연습 중인 곡 목록에서 선택
3. 연습 중: 특정 구간(마디)에 메모 (예: "왼손 4번 손가락 주의")
4. 연습 종료: 자동으로 연습 시간 기록, 컨디션 체크
5. 주말: 한 주간 연습 돌아보기, 다음 주 목표 설정

### 페르소나 4: 피아노 선생님 - 정하윤 (31세, 피아노 강사)

| 항목 | 내용 |
|------|------|
| **이름** | 정하윤 |
| **나이** | 31세 |
| **직업** | 피아노 학원 강사 (학생 20명 관리) |
| **피아노 경험** | 음대 졸업, 교습 경력 6년 |
| **사용 기기** | iPhone 15 Pro, MacBook |

**배경**
- 학생 20명의 레슨 진도와 과제를 관리해야 함
- 학생마다 진도, 실력, 연습량이 다 달라 개별 관리가 필요
- 현재 종이 출석부와 카카오톡 메모로 관리 중
- 학부모에게 아이의 진도와 과제를 알려줘야 하는 경우도 많음

**목표**
- 학생별 레슨 기록을 체계적으로 관리하고 싶음
- 학생의 연습량을 파악해서 레슨 방향을 조정하고 싶음
- 학부모에게 레슨 리포트를 간편하게 공유하고 싶음
- 반복적인 관리 업무 시간을 줄이고 싶음

**좌절점 (Pain Points)**
- 학생이 많아지면서 종이 기록의 한계를 느낌
- 학생이 과제를 안 해왔을 때 무엇을 냈는지 기억이 안 남
- 학부모 연락에 일일이 대응하기 번거로움
- 기존 피아노 앱은 학생 관리 기능이 없음

**앱 사용 시나리오**
1. 레슨 전: 학생의 이전 레슨 노트와 과제 확인
2. 레슨 중: 오늘 진도와 피드백을 앱에 바로 기록
3. 레슨 후: 다음 과제를 등록하고 학생에게 공유
4. 주간: 학생들의 연습량 통계를 확인하고 레슨 조정
5. 월말: 학부모에게 월간 리포트 발송

## 2. 사용자 여정 맵 (User Journey Map)

### 단계 1: 앱 발견 (Discovery)

| 요소 | 내용 |
|------|------|
| **행동** | 앱스토어에서 "피아노 연습" 검색, 블로그/SNS에서 추천 발견, 피아노 커뮤니티에서 입소문 |
| **감정** | 기대감 + 약간의 의심 ("또 다른 피아노 앱? 이게 다를까?") |
| **터치포인트** | 앱스토어 리스팅, 소셜미디어 광고, 블로그 리뷰, 지인 추천 |
| **기회 영역** | - 스크린샷과 설명에서 "연습 기록 + 레슨 관리"의 차별점을 명확히 전달 |
| | - "단순히 가르치는 앱이 아닌, 연습을 관리하는 앱"이라는 포지셔닝 |
| | - 실제 사용자 후기와 연습 통계 시각화를 앱스토어에 노출 |

### 단계 2: 온보딩 (Onboarding)

| 요소 | 내용 |
|------|------|
| **행동** | 앱 설치, 계정 생성, 피아노 경험 레벨 설정, 학습 목표 선택, 연습 시간 설정 |
| **감정** | 호기심 ("이 앱이 나한테 맞을까?") + 약간의 번거로움 |
| **터치포인트** | 온보딩 플로우, 프로필 설정, 튜토리얼 |
| **기회 영역** | - 3단계 이내의 간결한 온보딩 (경험 레벨, 목표, 연습 시간) |
| | - "독학자" / "레슨 수강생" / "재개자" 선택지로 맞춤 경험 제공 |
| | - 즉시 첫 연습을 시작할 수 있는 빠른 시작 옵션 |

### 단계 3: 첫 연습 기록 (First Practice Log)

| 요소 | 내용 |
|------|------|
| **행동** | 연습할 곡 추가, 타이머 시작, 구간별 메모, 연습 종료 및 기록 저장 |
| **감정** | 신선함 ("오 이렇게 기록하는 거구나!") + 약간의 귀찮음 ("매번 이렇게 해야 하나?") |
| **터치포인트** | 곡 추가 화면, 연습 타이머, 메모 에디터, 연습 요약 카드 |
| **기회 영역** | - 연습 시작/종료를 원탭으로 가능하게 (최소한의 마찰) |
| | - 첫 연습 기록 완료 시 축하 메시지와 가이드 |
| | - 곡 추가를 검색+자동완성으로 간편하게 |
| | - 연습 후 간단한 자기평가 (1~5 별점 또는 이모지) |

### 단계 4: 레슨 노트 작성 (Lesson Note)

| 요소 | 내용 |
|------|------|
| **행동** | 레슨 회차 생성, 선생님 피드백 기록, 과제 등록, 곡별 주의사항 메모 |
| **감정** | 뿌듯함 ("드디어 레슨 내용이 정리된다!") + 기대감 ("이걸 보면서 연습하면 효율적이겠다") |
| **터치포인트** | 레슨 노트 에디터, 과제 리스트, 곡-마디 연결, 사진/녹음 첨부 |
| **기회 영역** | - 레슨 노트 템플릿 제공 (곡명, 마디, 피드백, 과제 자동 구조화) |
| | - 이전 레슨과의 연결성 시각화 (진도 타임라인) |
| | - 음성 녹음 → 텍스트 변환으로 빠른 기록 |
| | - 선생님과 학생 간 공유 기능 |

### 단계 5: 습관 형성 (Habit Building)

| 요소 | 내용 |
|------|------|
| **행동** | 매일 연습 기록, 주간/월간 통계 확인, 목표 달성 확인, 연습 루틴 정착 |
| **감정** | 성취감 ("3주 연속 매일 연습했다!") / 좌절감 ("이번 주 하루도 못 했다...") |
| **터치포인트** | 연습 스트릭, 통계 대시보드, 알림, 주간 리포트 |
| **기회 영역** | - 연습 스트릭(연속 일수) + 캘린더 히트맵으로 동기부여 |
| | - 주간 리포트를 자동으로 생성하여 성장을 가시화 |
| | - 유연한 목표 설정 (일일 목표를 못 채워도 주간 목표로 보완) |
| | - 커뮤니티 기능으로 다른 학습자와 격려 (선택적) |
| | - 슬럼프 감지 시 격려 메시지와 작은 목표 제안 |

## 3. 경쟁 앱 분석

### 3.1 경쟁 앱 개요

| 앱 | 주요 대상 | 핵심 기능 | 가격 (연간) |
|----|----------|----------|------------|
| **Simply Piano** | 완전 초보, 어린이 | 마이크/MIDI 인식, 게이미피케이션, 5,000+ 곡 | $119~$210/년 |
| **Flowkey** | 초~중급, 곡 중심 학습 | 스플릿 스크린, Wait Mode, 1,500+ 곡 | $120/년 |
| **Piano Marvel** | 초~중급, 교육기관 | 시보 능력 평가(SASR), 25,000+ 곡, 교사 연동 | $111/년 |
| **Pianote** | 성인 독학자 | 고품질 비디오 레슨, 커뮤니티, 라이브 이벤트 | $200~$240/년 |
| **Modacity** | 모든 악기 연습자 | 연습 녹음, 시간 추적, 연습 저널 | 무료/$50/년 |
| **Better Practice** | 음악 교사-학생 | 과제 관리, 연습 추적, 스케줄 관리 | 교사 구독 모델 |

### 3.2 상세 분석

#### Simply Piano

**핵심 기능**
- 마이크 또는 MIDI 키보드로 실시간 음 인식
- 게이미피케이션 기반 학습 (레벨, 점수, 보상)
- 5,000곡 이상의 팝/클래식 라이브러리
- 가족 플랜 지원 (최대 5명)

**강점**
- 완전 초보에게 가장 친절한 UI/UX
- 아이들과 가족 사용에 최적화
- 브랜드 인지도가 높음

**약점**
- 테크닉(자세, 손 모양) 가이드 부족으로 나쁜 습관 형성 위험
- 마이크 음 인식 정확도 불안정
- 연습 기록/관리 기능 거의 없음
- 중급 이상으로 가면 콘텐츠 깊이 부족

#### Flowkey

**핵심 기능**
- 프로 피아니스트의 고품질 비디오 레슨
- Wait Mode (사용자가 맞는 음을 칠 때까지 대기)
- 속도 조절, 구간 반복, 좌/우손 분리 연습
- 난이도별 곡 분류 (초급~프로)

**강점**
- 곡 중심 학습에 최적화 (실제 곡을 빠르게 연주)
- 비디오 품질이 높고, 레슨이 간결
- 초~중급 학습자에게 가장 균형 잡힌 콘텐츠

**약점**
- 심화 이론과 테크닉 강의 부족
- 연습 기록/추적 기능 없음
- 레슨 관리, 과제 관리 기능 없음
- 완전 초보에게는 진입 장벽이 약간 있음

#### Piano Marvel

**핵심 기능**
- SASR (Standard Assessment of Sight Reading) 시보 능력 평가
- Hal Leonard, Alfred 등 공인 교재 기반 커리큘럼
- 25,000곡 이상의 방대한 라이브러리
- 교사-학생 연동 기능

**강점**
- 체계적인 커리큘럼 (클래식 교육에 강함)
- 정량적 실력 평가 시스템 (SASR)
- 교육기관에서 공식 채택하는 신뢰도
- 교사 관리 도구 일부 제공

**약점**
- UI가 다소 구식 (교육 도구 느낌이 강함)
- MIDI 키보드 필수 (마이크 인식은 베타)
- 모바일 경험이 데스크탑 대비 제한적
- 연습 습관 형성(스트릭, 알림 등) 기능 약함

#### Pianote

**핵심 기능**
- 10단계 체계적 커리큘럼 (The Method)
- 프로 강사의 고품질 비디오 레슨
- 활발한 학습자 커뮤니티
- 라이브 스트리밍 이벤트, 영상 피드백 (Plus 플랜)

**강점**
- 강사 퀄리티가 업계 최고 수준
- 성인 학습자에 최적화된 콘텐츠 톤
- 커뮤니티가 활발하고 동기부여 효과 큼
- 팝/록/블루스/재즈 등 다양한 장르

**약점**
- 가격이 높은 편 ($200~$240/년)
- Level 4 이후 학습 경로가 비선형적이라 혼란 가능
- 피드백 응답 시간이 느림 (수일~수주)
- 연습 추적/기록 기능 없음
- 클래식 학습에는 부적합

#### Modacity

**핵심 기능**
- 연습 시간 추적 및 타이머
- 연습 녹음 및 재생
- 연습 영역별 목표 설정
- 연습 통계 대시보드

**강점**
- 연습 기록에 특화된 거의 유일한 앱
- 모든 악기 지원 (피아노 전용이 아님)
- 녹음 비교 기능으로 성장 확인 가능

**약점**
- 피아노 학습 콘텐츠 없음 (순수 기록 도구)
- 레슨 노트/과제 관리 기능 없음
- 사용자 기반이 작고 업데이트 빈도가 낮음
- 한국어 미지원

### 3.4 차별화 포인트

기존 앱들은 **"가르치는 것"** 에 집중하지만, 우리 앱은 **"연습을 관리하고 레슨을 기록하는 것"** 에 집중합니다.

| 차별화 영역 | 기존 앱의 한계 | 우리 앱의 접근 |
|------------|--------------|--------------|
| **연습 기록** | 대부분 연습 기록 기능 없음 (Modacity만 있지만 피아노 비특화) | 피아노 전용 연습 기록 (곡별, 구간별, 시간 추적) |
| **레슨 노트** | 레슨 기록 기능이 있는 피아노 앱 없음 | 회차별 레슨 노트, 과제 관리, 피드백 기록 |
| **습관 형성** | 게이미피케이션 위주 (지속성 낮음) | 스트릭, 캘린더 히트맵, 유연한 목표, 주간 리포트 |

## 4. 핵심 인사이트

### 4.1 사용자 니즈 요약

| 우선순위 | 니즈 | 대상 |
|---------|------|------|
| **P0** | 연습 기록을 간편하게 남기고 싶다 | 모든 사용자 |
| **P0** | 레슨에서 배운 내용을 체계적으로 정리하고 싶다 | 레슨 수강생 |
| **P0** | 꾸준히 연습하는 습관을 만들고 싶다 | 모든 사용자 |
| **P1** | 곡별/구간별 연습 진도를 추적하고 싶다 | 모든 사용자 |
| **P1** | 선생님이 학생의 연습 현황을 파악하고 싶다 | 교사 |
| **P1** | 연습의 성장을 수치/시각으로 확인하고 싶다 | 모든 사용자 |
| **P2** | 다른 학습자와 경험을 나누고 싶다 | 독학자, 재개자 |
| **P2** | 학부모에게 아이의 진도를 공유하고 싶다 | 교사 |

### 4.2 주요 페인포인트

1. **연습 기록의 부재**: 현재 피아노 학습자 대부분이 연습 기록을 남기지 않거나, 종이/메모 앱에 비체계적으로 기록. 기존 피아노 앱은 "가르치기"에만 집중하고, "연습 관리"는 방치된 영역

2. **레슨 내용 망각**: 주 1회 레슨에서 선생님이 말해준 세부 피드백(템포, 손가락 번호, 강약, 페달링 등)을 다음 연습 때 기억하지 못함. 이로 인해 같은 피드백을 반복하게 되어 레슨 효율 저하

3. **동기부여 유지 실패**: 성인 학습자의 가장 큰 장벽은 "지속"임. 바쁜 일상에서 연습이 밀리면 죄책감 → 회피 → 포기의 악순환. 게이미피케이션만으로는 장기 지속이 어려움

4. **인지-운동 괴리 (Cognitive-Motor Gap)**: 성인 학습자는 이해는 빠르지만 손이 따라오지 않아 좌절. "왜 이렇게 쉬운 게 안 될까?"라는 자기 비판이 동기를 떨어뜨림

5. **체계적 경로 부재**: 독학자는 "다음에 뭘 해야 하지?"라는 막막함. 재개자는 "내 레벨이 어디지?"라는 혼란. 레슨 수강생도 레슨 사이 자율 연습 가이드가 부족

### 4.3 디자인 기회 영역

#### 기회 1: "마찰 없는 연습 기록"
- 연습 시작/종료를 원탭으로
- 자동 시간 측정 + 최소한의 수동 입력
- 연습 후 30초 내에 기록 완료 가능한 플로우
- **핵심 원칙**: 기록하는 행위 자체가 부담이 되면 안 됨

#### 기회 2: "구조화된 레슨 노트"
- 템플릿 기반 레슨 노트 (곡명 → 마디 → 피드백 구조)
- 이전 레슨과의 연결성 자동 생성
- 음성 메모, 사진 첨부로 빠른 기록
- 교사-학생 간 노트 공유 및 과제 관리
- **핵심 원칙**: 레슨 직후 2분 이내에 핵심 내용 기록 가능

#### 기회 3: "지속 가능한 동기부여 시스템"
- 연습 스트릭 + 캘린더 히트맵 (GitHub 커밋 그래프처럼)
- 유연한 목표 (일일 목표를 못 채워도 주간으로 보완)
- 주간 자동 리포트 (이번 주 N시간 연습, N곡 진행)
- 슬럼프 감지 시 부드러운 격려 메시지
- **핵심 원칙**: 죄책감이 아닌 성취감 기반 동기부여

#### 기회 4: "피아노 교사 도구"
- 학생별 레슨 히스토리 & 연습량 대시보드
- 원탭 과제 배정 및 진행 상황 추적
- 학부모 리포트 자동 생성 및 공유
- **핵심 원칙**: 교사의 관리 부담을 줄이면 학생 경험도 개선됨

#### 기회 5: "한국 피아노 교육 맞춤"
- 한국에서 많이 사용되는 교재(체르니, 하농, 부르크뮐러 등) 기반 진도 추적
- 한국 피아노 학원 문화에 맞는 회차 관리
- 한국어 네이티브 UX
- **핵심 원칙**: 글로벌 앱이 놓치는 로컬 니즈를 정확히 공략
