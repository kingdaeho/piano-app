## 1. 제품 비전 및 목표

### 1.1 제품 비전

피아노를 배우는 사람들이 체계적으로 연습을 기록하고, 레슨 내용을 관리하며, 자신의 성장 과정을 시각적으로 확인할 수 있는 개인 피아노 학습 관리 앱.

### 1.2 해결하려는 문제

| 문제 | 현재 상황 | 목표 |
|------|----------|------|
| 연습 기록 부재 | 연습 시간과 내용을 기억에 의존 | 일별/주별 연습 데이터를 체계적으로 기록 |
| 레슨 내용 망각 | 레슨 후 선생님 피드백을 노트에 적거나 기억에 의존 | 회차별 레슨 노트와 피드백을 디지털로 관리 |
| 진도 파악 어려움 | 어떤 곡을 얼마나 진행했는지 감으로 판단 | 곡별 진행도와 난이도를 시각적으로 추적 |
| 동기 부여 부족 | 성장을 체감하기 어려워 연습 의지 저하 | 목표 설정과 달성률 시각화로 동기 부여 |
| 학습 여정 기록 없음 | 장기적인 성장 과정을 돌아보기 어려움 | 타임라인으로 학습 여정과 마일스톤 시각화 |

### 1.3 핵심 가치 제안

- **체계적 관리**: 흩어진 연습/레슨 정보를 한 곳에서 관리
- **성장 시각화**: 데이터 기반으로 자신의 성장을 눈으로 확인
- **지속적 동기 부여**: 목표와 달성률을 통해 꾸준한 연습 유도
- **레슨 효율 극대화**: 레슨 내용을 체계적으로 기록하여 복습 효율 향상

## 2. 타겟 사용자

### 2.1 주요 사용자 (Primary)

**피아노 학습자 (성인 취미반)**
- 연령: 20~50대
- 피아노 학원이나 개인 레슨을 받는 성인
- 주 1~2회 레슨, 매일 또는 주 3~5회 자가 연습
- 디지털 도구 사용에 익숙함
- 체계적인 연습 관리에 대한 니즈가 높음

### 2.2 부차 사용자 (Secondary)

**피아노 학습자 (학생/입시생)**
- 연령: 10~20대
- 정기적으로 레슨을 받으며 집중적으로 연습
- 곡별 완성도와 진행 상황 추적이 중요

### 2.3 사용자 페르소나

**페르소나 1: 민지 (32세, 직장인)**
- 6개월 전부터 피아노 학원 수강
- 주 1회 레슨, 퇴근 후 하루 30분~1시간 연습
- 레슨에서 배운 내용을 자주 잊어버림
- 연습을 꾸준히 하고 싶지만 동기 부여가 어려움

**페르소나 2: 준혁 (45세, 자영업자)**
- 1년 전부터 피아노를 독학 + 월 2회 개인 레슨
- 시간이 불규칙하여 연습 패턴이 일정하지 않음
- 여러 곡을 동시에 연습하며 각 곡의 진행도를 파악하고 싶음

## 3. 핵심 기능 상세 설명

### 3.1 연습 기록 (Practice Log)

사용자가 일별/주별 연습 시간과 내용을 기록하고 추적할 수 있는 기능.

**상세 요구사항:**

| ID | 요구사항 | 설명 |
|----|----------|------|
| PL-01 | 연습 세션 기록 | 시작/종료 시간, 연습한 곡, 연습 내용(메모) 기록 |
| PL-02 | 타이머 기능 | 연습 시작 시 타이머를 실행하여 자동으로 시간 측정 |
| PL-03 | 일별 연습 요약 | 해당 날짜의 총 연습 시간, 연습한 곡 목록 표시 |
| PL-04 | 주별 연습 통계 | 주간 총 연습 시간, 일별 분포, 곡별 연습 시간 차트 |
| PL-05 | 월별 연습 캘린더 | 캘린더 뷰에서 연습한 날짜와 연습량 한눈에 확인 |
| PL-06 | 곡별 연습 시간 집계 | 특정 곡에 투자한 총 연습 시간 누적 표시 |

### 3.2 레슨 노트 (Lesson Notes)

회차별 레슨 내용과 선생님 피드백을 기록하고 관리하는 기능.

**상세 요구사항:**

| ID | 요구사항 | 설명 |
|----|----------|------|
| LN-01 | 레슨 회차 관리 | 레슨 날짜, 회차 번호, 레슨 시간 기록 |
| LN-02 | 레슨 내용 기록 | 자유 텍스트로 레슨에서 배운 내용 기록 |
| LN-03 | 선생님 피드백 | 선생님이 지적한 사항, 개선점 별도 섹션으로 기록 |
| LN-04 | 과제/숙제 관리 | 다음 레슨까지 연습할 내용, 과제 체크리스트 |
| LN-05 | 레슨별 연관 곡 태그 | 레슨에서 다룬 곡을 태그로 연결 |
| LN-06 | 이미지 첨부 | 악보 사진, 손 자세 사진 등 이미지 첨부 기능 |
| LN-07 | 레슨 히스토리 | 전체 레슨 목록을 시간순으로 조회 |

### 3.3 곡 관리 (Repertoire Management)

연습 중인 곡 목록을 관리하고 각 곡의 진행 상태를 추적하는 기능.

**상세 요구사항:**

| ID | 요구사항 | 설명 |
|----|----------|------|
| RM-01 | 곡 등록 | 곡명, 작곡가, 난이도, 장르 등 기본 정보 등록 |
| RM-02 | 진행 상태 관리 | 상태 구분: 시작 전 / 연습 중 / 마무리 단계 / 완성 / 보류 |
| RM-03 | 난이도 표시 | 사용자가 체감하는 난이도 (1~5 레벨) |
| RM-04 | 진행도 퍼센트 | 곡의 완성도를 퍼센트로 표시 (수동 입력) |
| RM-05 | 구간별 메모 | 곡의 특정 마디/구간에 대한 연습 메모 |
| RM-06 | 곡별 연습 기록 연동 | 해당 곡의 연습 세션 이력 자동 연결 |
| RM-07 | 곡 목록 필터/정렬 | 상태별, 난이도별, 최근 연습순 등으로 필터/정렬 |

### 3.4 목표 설정 (Goal Setting)

일일/주간 연습 목표를 설정하고 달성률을 시각적으로 확인하는 기능.

**상세 요구사항:**

| ID | 요구사항 | 설명 |
|----|----------|------|
| GS-01 | 일일 연습 목표 | 하루 목표 연습 시간 설정 (예: 60분) |
| GS-02 | 주간 연습 목표 | 주간 목표 연습 일수/시간 설정 |
| GS-03 | 달성률 표시 | 오늘/이번 주 목표 대비 달성률 퍼센트 및 프로그레스 바 |
| GS-04 | 연속 연습 스트릭 | 연속으로 목표를 달성한 일수 (Streak) 표시 |
| GS-05 | 곡 완성 목표 | 특정 곡을 목표 날짜까지 완성하겠다는 목표 설정 |
| GS-06 | 목표 알림 | 연습 시간이 되면 알림 (선택 사항) |

### 3.5 타임라인 (Learning Timeline)

학습 여정을 시각적으로 보여주는 타임라인과 마일스톤 기록 기능.

**상세 요구사항:**

| ID | 요구사항 | 설명 |
|----|----------|------|
| TL-01 | 학습 타임라인 뷰 | 시간순으로 주요 이벤트를 타임라인 형태로 시각화 |
| TL-02 | 마일스톤 기록 | 첫 곡 완성, 레슨 100회차 등 주요 성취 기록 |
| TL-03 | 자동 이벤트 생성 | 곡 완성, 레슨 기록, 목표 달성 시 자동으로 타임라인에 추가 |
| TL-04 | 수동 이벤트 추가 | 발표회 참가, 자격증 취득 등 사용자 정의 이벤트 추가 |
| TL-05 | 월별/연별 요약 | 특정 기간의 학습 성과를 요약하여 표시 |

## 5. 사용자 스토리

### 5.1 연습 기록

```
US-01: 연습 세션 시작
As a 피아노 학습자
I want to 연습 시작 시 타이머를 누르면 자동으로 시간이 측정되도록
So that 별도로 시간을 기록하지 않아도 정확한 연습 시간이 기록된다.

인수 조건:
- 연습 시작 버튼을 누르면 타이머가 시작된다.
- 연습 중 곡을 선택할 수 있다.
- 연습 종료 시 총 연습 시간과 곡별 시간이 저장된다.
- 연습 중 메모를 남길 수 있다.
```

```
US-02: 주간 연습 현황 확인
As a 피아노 학습자
I want to 이번 주 얼마나 연습했는지 한눈에 확인하고 싶다
So that 연습 패턴을 파악하고 부족한 부분을 보완할 수 있다.

인수 조건:
- 주간 총 연습 시간이 표시된다.
- 일별 연습 시간 막대 차트가 표시된다.
- 곡별 연습 시간 비율이 표시된다.
- 지난 주 대비 증감이 표시된다.
```

### 5.2 레슨 노트

```
US-03: 레슨 후 노트 작성
As a 피아노 학습자
I want to 레슨 직후 배운 내용과 선생님 피드백을 빠르게 기록하고 싶다
So that 레슨 내용을 잊지 않고 다음 연습에 반영할 수 있다.

인수 조건:
- 레슨 날짜와 회차가 자동으로 설정된다.
- 레슨 내용을 자유롭게 텍스트로 작성할 수 있다.
- 선생님 피드백을 별도 섹션에 기록할 수 있다.
- 다음 레슨까지의 과제를 체크리스트로 작성할 수 있다.
- 레슨에서 다룬 곡을 태그할 수 있다.
```

```
US-04: 이전 레슨 내용 복습
As a 피아노 학습자
I want to 이전 레슨에서 지적받은 사항을 연습 전에 빠르게 확인하고 싶다
So that 같은 실수를 반복하지 않고 효율적으로 연습할 수 있다.

인수 조건:
- 최근 레슨 노트가 메인 화면에서 바로 접근 가능하다.
- 선생님 피드백만 필터링하여 볼 수 있다.
- 특정 곡에 대한 피드백만 모아볼 수 있다.
```

### 5.3 곡 관리

```
US-05: 연습 곡 등록 및 관리
As a 피아노 학습자
I want to 현재 연습 중인 곡들을 한눈에 보고 각 곡의 진행 상태를 관리하고 싶다
So that 여러 곡의 연습 우선순위를 정하고 균형 있게 연습할 수 있다.

인수 조건:
- 곡명, 작곡가, 난이도를 입력하여 곡을 등록할 수 있다.
- 곡의 진행 상태를 변경할 수 있다 (시작 전/연습 중/마무리/완성/보류).
- 곡 목록을 상태별, 난이도별로 필터링할 수 있다.
- 각 곡에 대한 총 연습 시간이 표시된다.
```

### 5.4 목표 설정

```
US-06: 일일 연습 목표 설정 및 확인
As a 피아노 학습자
I want to 하루 연습 목표를 설정하고 달성률을 실시간으로 확인하고 싶다
So that 매일 꾸준히 연습하는 습관을 만들 수 있다.

인수 조건:
- 일일 목표 연습 시간을 설정할 수 있다.
- 오늘의 달성률이 프로그레스 바로 표시된다.
- 연속 달성 일수(스트릭)가 표시된다.
- 목표 달성 시 축하 메시지가 표시된다.
```

### 5.5 타임라인

```
US-07: 학습 여정 돌아보기
As a 피아노 학습자
I want to 피아노를 배우기 시작한 이후의 성장 과정을 타임라인으로 돌아보고 싶다
So that 성장을 체감하고 지속적인 동기 부여를 받을 수 있다.

인수 조건:
- 곡 완성, 레슨 기록 등 주요 이벤트가 타임라인에 표시된다.
- 마일스톤(첫 곡 완성, 50일 연속 연습 등)이 강조 표시된다.
- 월별/연별로 타임라인을 탐색할 수 있다.
```

## 부록: MVP 스코프 요약

**MVP (Phase 1)** 에 포함되는 기능:

1. 연습 기록: 타이머 기반 연습 세션 기록, 일별/주별 통계
2. 레슨 노트: 회차 관리, 내용 기록, 선생님 피드백
3. 곡 관리: 곡 등록, 진행 상태 관리
4. 목표 설정: 일일 목표, 달성률 표시

2. 과제 관리, 레슨-곡 태그 연결
3. 난이도, 진행도 퍼센트, 필터/정렬
4. 주간 목표, 연속 스트릭

**Phase 3** 에서 추가:

1. 이미지 첨부, 구간별 메모
2. 타임라인 뷰, 마일스톤
3. 곡 완성 목표
